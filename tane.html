種の工房

図のラフとネタを直感的に置いて、Claude Code に渡せる作図プロンプトに変換します。最終的な規約準拠の SVG は、受け取った Claude Code が清書します。

1. ネタ（言葉）

作法：身体感覚・日常の場面に接続する／値段・権威・経済格差に逃げない／良い側を言い切り、対比は図に任せる。

2. ラフ作図（直感的に置く）

320×120 のキャンバス。要素を足してドラッグで配置。角の白い四角でリサイズ。精度は不要、意図が伝わればよい。

要素を選ぶと、ここで色・文字・意図メモを編集できます。
⚠ accent（朱）が2つ以上あります。規約では「accentは1図に1箇所まで」。1つに絞ってください。
